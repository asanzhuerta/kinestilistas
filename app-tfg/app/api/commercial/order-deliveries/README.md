# API repartos comercial

Endpoints para preparar y consultar repartos derivados de pedidos confirmados.

## Función

Permitir que el comercial agrupe líneas de un pedido en uno o varios repartos, indique manualmente los bultos y genere la etiqueta logística correspondiente.

Cada reparto tiene estado propio y puede ser de tipo `commercial` o `agency`. Los repartos comerciales se vinculan a visitas y validación QR; los repartos de agencia generan una etiqueta diferenciada.
