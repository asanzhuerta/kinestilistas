# API pagos pedido comercial

Endpoint para registrar pagos parciales o totales sobre un pedido accesible por el comercial.

## Función

Guardar pagos asociados a un pedido entregado, con importe, método, fecha, notas y usuario que registra la operación.

El historial de pagos permite calcular saldo pendiente y mantener trazabilidad para informes posteriores.
