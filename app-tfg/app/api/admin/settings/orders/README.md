# API ajustes de pedidos

Endpoint administrativo para parámetros de negocio del módulo de pedidos.

## Función

Leer y actualizar ajustes operativos que afectan al cálculo de pedidos, empezando por el cargo de entrega por agencia.

Los valores se guardan en configuración persistente para evitar literales fijos en la lógica de pedido.
