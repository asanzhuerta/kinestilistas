# Visitas comercial

Gestion y detalle de visitas.

## Funcion

Planificar, consultar y actualizar visitas a clientes.

Reglas recientes a tener presentes:

- una visita de reparto aplazada libera los repartos asociados para permitir su reasignacion posterior;
- las ETA visibles en listados y mapa reutilizan la misma planificacion diaria que consume la vista de rutas;
- los formularios largos en modal deben usar el patron comun scrollable para no romper la experiencia movil.
