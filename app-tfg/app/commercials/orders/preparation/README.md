# Preparación de repartos

Pantalla comercial para convertir pedidos confirmados en repartos preparados.

## Función

Seleccionar qué líneas y cantidades de un pedido entran en un reparto concreto, registrar manualmente el número de bultos y descargar la etiqueta de reparto.

Esta ruta separa el pedido comercial del acto logístico de entrega, porque un mismo pedido puede necesitar varios repartos si no todos los productos están disponibles al mismo tiempo.
