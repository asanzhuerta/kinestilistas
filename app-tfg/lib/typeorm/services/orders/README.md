# Servicios pedidos

Operaciones persistentes de pedidos.

## Función

Gestionar borradores, líneas, confirmaciones, historial, estados, repartos preparados, pagos parciales y ajustes de entrega.

Los servicios de esta carpeta son la fuente de verdad del módulo M4: aplican promociones, bloqueos de negocio, separación entre pedido y reparto, cargo de agencia y actualización del estado de cobro según el historial de pagos.
