# Integraciones

Preparación de conectores y estados de integración con servicios externos.

## Función

Centralizar lógica vinculada a automatizaciones, sistemas externos y futuros flujos como Factusol/n8n.

El módulo M7 aspira a integrar procesos externos. Esta carpeta reserva ese dominio sin mezclarlo con operaciones internas.

La preparación de Factusol mediante n8n empieza en `factusol-n8n.ts`, donde se concentran el nombre estable de la integración, los tipos de datos previstos y la configuración base. La integración real queda pendiente de credenciales, migraciones específicas y definición del flujo de negocio con Factusol.
